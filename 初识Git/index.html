Git仓库

  • workspace ：工作区
  • index / stage : 暂存区
  • repository : 本地仓库
  • remote : 远程仓库

文件状态 ：对于任何一个文件，Git内部只有三个状态 “ 已提交（committed）,已修改（modified），已暂存（staged）”，已提交表示文件已经被安全地保存在本地数据库中，已修改表示修改了某个文件，但是还没有提交保存，已暂存表示已经修改的文件放在下次提交时要保存的清单中
